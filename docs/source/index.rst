====
aspy
====

``aspy`` is a Pure Python Power Systems Simulator.
The latest development version can be found `here <https://github.com/dioph/aspy>`_.

Installation
============
``aspy`` requires
``numpy`` (for linear algebra),
``PyQt5`` (for the GUI),
``pylatex`` (to generate reports),
``matplotlib`` (for plotting), and
``networkx`` (for multigraph data structures)

Installing the most recent stable version of the package is as easy as::

    pip install aspy
